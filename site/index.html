Интерактивные приложения (работают прямо в браузере):

Python Calculator (Калькулятор на Python)

Описание: Простой калькулятор на Python с интерактивным меню.

Docker команды:

docker build -t calc-python calc-py/
docker run -it --name my-calc-python calc-python

Java Calculator (Калькулятор на Java)

Описание: Калькулятор на Java для арифметических операций.

Docker команды:

docker build -t calc-java calc-java/
docker run -it --name my-calc-java calc-java

Dart Calculator (Калькулятор на Dart)

Описание: Калькулятор на Dart для арифметических операций.

Docker команды:

docker build -t calc-dart dart-calc/
docker run -it --name my-calc-dart calc-dart

Java Guessing Game (Игра "Угадай число" на Java)

Описание: Игра на Java, где компьютер загадывает число от 1 до 100.

Docker команды:

docker build -t game-java game-java/
docker run -it --name my-game-java game-java

Rock Paper Scissors Game (Игра "Камень-ножницы-бумага" на Python)

Описание: Классическая игра на Python, где игрок играет против компьютера.

Docker команды:

docker build -t rpsgame-python rpsgame-py/
docker run -it --name my-rpsgame rpsgame-python

Static HTML Website (Веб-сайт на Nginx)

Описание: Этот сайт! Статический HTML-сайт, раздаваемый через nginx в Docker контейнере.

Docker команды:

docker build -t my-website site/
docker run -d -p 8080:80 --name my-site my-website

О проекте

Этот проект демонстрирует создание собственных Dockerfile для различных типов приложений.

Цели работы:

  • Написать собственные Dockerfile для программ на разных языках
  • Собрать Docker контейнеры
  • Загрузить проект на GitHub
  • Документировать процесс работы

Технологии:

  • Python: калькулятор и игра "Камень-ножницы-бумага"
  • Java: калькулятор и игра "Угадай число"
  • Dart: калькулятор
  • HTML/JavaScript/Nginx: этот веб-сайт
