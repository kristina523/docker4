Практическая работа №4 - Собственные Dockerfile

Добро пожаловать! Здесь представлены различные программы, которые контейнеризованы с помощью Docker.

1. Python Calculator Python

Описание: Простой калькулятор на Python с интерактивным меню для выполнения арифметических операций (сложение, вычитание, умножение, деление).

Команды сборки и запуска:

docker build -t calc-python calc-py/
docker run -it --name my-calc-python calc-python

2. Java Calculator Java

Описание: Калькулятор на Java, выполняющий базовые арифметические операции с двумя числами.

Команды сборки и запуска:

docker build -t calc-java calc-java/
docker run -it --name my-calc-java calc-java

3. Dart Calculator Dart

Описание: Калькулятор на Dart для выполнения арифметических операций.

Команды сборки и запуска:

docker build -t calc-dart dart-calc/
docker run -it --name my-calc-dart calc-dart

4. Java Guessing Game Java

Описание: Игра "Угадай число" на Java, где компьютер загадывает число от 1 до 100, а пользователь пытается его угадать.

Команды сборки и запуска:

docker build -t game-java game-java/
docker run -it --name my-game-java game-java

5. Rock Paper Scissors Game Python

Описание: Классическая игра "Камень-ножницы-бумага" на Python, где игрок играет против компьютера.

Команды сборки и запуска:

docker build -t rpsgame-python rpsgame-py/
docker run -it --name my-rpsgame rpsgame-python

6. Static HTML Website Nginx

Описание: Этот сайт! Статический HTML-сайт, раздаваемый через nginx в Docker контейнере.

Команды сборки и запуска:

docker build -t my-website site/
docker run -d -p 8080:80 --name my-site my-website
