О проекте

Практическая работа №4 - Docker

Этот проект демонстрирует создание собственных Dockerfile для различных типов приложений.

Цели работы:

  • Написать собственные Dockerfile для программ на разных языках
  • Собрать Docker контейнеры
  • Загрузить проект на GitHub
  • Документировать процесс работы

Технологии:

  • Python: калькулятор и игра "Камень-ножницы-бумага"
  • Java: калькулятор и игра "Угадай число"
  • Dart: калькулятор
  • HTML/CSS/Nginx: этот веб-сайт
